Listas

En HTML tenemos las listas desordenadas (ul) unordered lists, las listas ordenadas (ol) ordered lists, ambas a su vez llevan etiquetas hijas dentro de ellas, que son los li, los cuales contienen los items o elementos a listar, el uso de una o de otra dependera de lo que necesitemos listar, y si el orden importa o no importa.

Lista ordenada

  1. Elemento 01
  2. Elemento 02
  3. Elemento 03

Lista desordenada

  • Elemento 01
  • Elemento 02
  • Elemento 03
