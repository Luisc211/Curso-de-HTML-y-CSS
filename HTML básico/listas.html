Listas

En HTML tenemos 2 tipos de listas, las OL (Ordered List), o en español "lista ordenada", y las UL (Unordered list), o en español "Lista desordenada", el uso de una o de otra dependera de los elementos que necesitemos o deseemos listar, sí el orden de los elementos importan debemos utilizar OL, cada una a su vez contiene etiquetas hijas llamadas li sí el orden de los elementos no es relevante, podemos utilizar UL, a continuación se mostraran un ejemplo para cada una

Ejemplo de una llista ordenada

  1. Paso 1
  2. Paso 2
  3. Paso 3

Ejemplo de una lista desordenada

  • Nosotros
  • Contacto
  • Nuestra historia
